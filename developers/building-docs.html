Build this Documentation

Serve Docs Locally

Install npm

apt-get install npm

Install gitbook

npm install gitbook-cli -g

Clone Repo

git clone path/to/repo

Install Plugins

cd path/to/repo
gitbook install

Serve

cd path/to/repo
gitbook serve

Browse

The documentation is served at localhost:4000 by default.
